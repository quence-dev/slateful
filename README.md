# Slateful

AI-powered slate detection and clip renaming plugin for Adobe Premiere Pro.

## Features (Planned)

- OCR-based slate reading
- Manual slate marking (v1.0)
- Automatic slate detection (v2.0)
- Customizable naming templates
- 100% local processing (no API keys required)

## Status

**In Development** - v1.0 MVP

Currently building the manual slate marking feature.
